1) قاعدة الـ API (Backend / REST)
--------------------------------------------------------------------------------
المتغير: VITE_API_BASE_URL
الافتراضي في الكود إن لم يُضبط المتغير:
  http://localhost:8000/api

كل المسارات أدناه تُلحق بهذه القاعدة (بدون شرطة مزدوجة إضافية).
مثال كامل: POST  http://localhost:8000/api/auth/login

الرؤوس (محمي): Authorization: Bearer <token>
(يُضبط تلقائياً من axiosInstance عند وجود توكن)

--------------------------------------------------------------------------------
2) مسارات مستخدمة فعلياً في الكود (axios)
--------------------------------------------------------------------------------

[AUTH — مصادقة]
  POST   /auth/login
         Body JSON: { email, password, role }  — role: patient | doctor | admin
         استخدام: src/features/auth/authSlice.js

  POST   /auth/register
         Body: multipart/form-data (نموذج التسجيل)
         استخدام: src/features/auth/authSlice.js

[APPOINTMENTS — مواعيد المريض]
  GET    /appointments
         استخدام: src/features/appointments/appointmentsSlice.js

  DELETE /appointments/:id
         :id = معرف الموعد
         استخدام: src/features/appointments/appointmentsSlice.js

[INVOICES — فواتير المريض]
  GET    /invoices
         استخدام: src/features/invoices/invoicesSlice.js

  POST   /invoices/:invoiceId/pay
         Body JSON: { paymentMethod, cardData? }
         استخدام: src/features/invoices/invoicesSlice.js

[MEDICAL_RECORDS — السجل الطبي للمريض]
  GET    /medical-records
         استخدام: src/features/medicalRecords/medicalRecordsSlice.js

  PUT    /medical-records
         Body: بيانات التحديث (حسب العقد)
         استخدام: src/features/medicalRecords/medicalRecordsSlice.js

[ADMIN — إدارة (جزء مستخدم)]
  POST   /admin/doctors
         استخدام: src/Pages/admin/AdminDoctorsPage.jsx

  POST   /admin/admins
         استخدام: src/Pages/admin/AdminStaffPage.jsx
